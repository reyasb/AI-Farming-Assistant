#🌱 AI Farming Assistant for Kerala Farmers

📌 Project Overview
AI Farming Assistant is a beginner-friendly web application developed to help Kerala farmers make better farming decisions using Artificial Intelligence.

The system collects farm information such as soil type, weather conditions, crop details, and nutrient values and provides AI-based recommendations.

🚀 Features
- Crop Recommendation
- Fertilizer Recommendation
- Weather Advice
- Disease Prevention Suggestions
- Farmer Information Collection Form
- Kerala-specific agricultural guidance

🛠️ Technologies Used
#Frontend
- HTML
- CSS
- Bootstrap
- JavaScript

Backend
- Node.js
- Express.js

AI Integration
- OpenAI API

📋 Input Parameters
- District
- Soil Type
- Crop Name
- Season
- Temperature
- Humidity
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Soil pH

📊 Output
The AI generates:
1. Suitable crop recommendations
2. Fertilizer suggestions
3. Weather precautions
4. Disease prevention advice

🎯 Target Users
- Kerala Farmers
- Agricultural Students
- Small Scale Farming Communities
